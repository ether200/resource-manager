## Description

My first project using the MERN stack. It was done to serve the purpose of managing my educational resources videos. It supports the CRUD operations, account registry, form validation, video player.
#### Technologies



- Express
- JWT
- MongoDB
- NodeJS
- React-Router
- React-Hook-Form
- Redux
- Typescript
